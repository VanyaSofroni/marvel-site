# Сайт Marvel

### Как использовать

1. ```clone this repo```  
2. ```cd path/to/...```   переходим в папку с проектом
3. ```npm i```            установка зависимостей
4. ```gulp build```       сборка проекта
5. ```gulp watch```       разработка проекта

[Посмотреть](https://vanyasofroni.github.io/marvel-site/)
